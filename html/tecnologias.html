Tecnologias Utilizadas

Este proyecto fue realizado con HTML, CSS y JAVASCRIPT como base.las animaciones fueron creadas apartir de anime.js, ScrollReveal.js, se utilizo BOOTSTRAP para la creación de las cards de los pokemones y Googlefonts para todas las letras. Se utilizaron en su mayoria svg para poder dar mas adaptamiento a las imagenes, dentro del footer se puede encontrar a Pikachu el cual esta realizado pixel por pixel con Html y CSS.

Se innovó, proporcionando Cards de pokemones dinámicas, selección de pokemones por nombre, paginación del catalogo y mostrando información breve sobre cada elemento al cual se posicione el cursor del usuario. Los recursos de imagenes son proporcionadas por una api externa y la estructura de datos de la aplicación fué generada a partir de un conjunto de solicitudes a la pokeapi.
